
# HRNet

## 数据集
**Person Keypoints 数据集**

- 包含 17 个人体关键点，数据来源于 COCO 2017。
- 裁剪出单人图像及其对应的关键点。

### 数据集规模
- **训练集**: 13043 张图片
- **验证集**: 561 张图片
